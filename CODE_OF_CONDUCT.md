# Code of Conduct

## Our standard
This project is committed to a respectful, welcoming, and harassment-free experience for everyone.

## Expected behavior
- Be respectful and constructive.
- Accept feedback with patience.
- Focus on ideas, not people.
- Help keep discussions clear and useful.

## Unacceptable behavior
- Harassment, insults, or personal attacks
- Discrimination of any kind
- Publishing private information without permission
- Repeated disruptive or bad-faith behavior

## Scope
This Code of Conduct applies in project spaces, issue threads, pull requests, and related discussions.

## Reporting
If you see behavior that violates this Code of Conduct, open a private issue with details so maintainers can review and respond.

## Enforcement
Project maintainers may remove, edit, or reject comments and contributions that violate this policy.
